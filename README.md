# Газетный чанкер (Newspaper Chunker)

Инструмент для разбиения газетных публикаций на смысловые чанки с извлечением ключевых элементов (дат, имен, организаций, событий).

## Возможности

- Рекурсивный обход папок с документами
- Поддержка форматов: PDF, DOC, DOCX, TXT, RTF
- Извлечение текста через Docker + pdftotext
- Смысловое разбиение на чанки (20-300 символов)
- Выделение ключевых элементов:
  - Даты и временные периоды
  - Имена людей
  - Организации и бренды
  - Географические названия
  - Культурные объекты
  - Цитаты и лозунги
  - Числовые показатели
- Сохранение результатов в JSON и TXT

## Требования

- Windows 10/11
- Python 3.8+
- Docker Desktop
- PowerShell 5.0+
